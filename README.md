**ConMigo** es una aplicación móvil diseñada para facilitar el aprendizaje. Inspirada en dinámicas de gamificación, busca aprender con  comunicación de manera interactiva y accesible.

---

---

---

##  Características Principales
* **Mapa de Aprendizaje:** Unidades temáticas con progresión lógica.
* **Sistema de Energía:** Gestión de vidas para incentivar el estudio constante.
* **Interactividad Avanzada:** 
  * Reconocimiento de voz (STT) para práctica bilingüe.
  * Síntesis de voz (TTS) para apoyo auditivo.
  * Ejercicios de parejas y selección múltiple.
* **Sincronización Cloud:** Persistencia de datos en tiempo real con Firebase.

---

## 🛠️ Stack Tecnológico
* **Frontend:** Flutter SDK
* **Estado:** Pattern BLoC (Business Logic Component)
* **Backend:** Firebase (Auth, Firestore, Storage)
* **Animaciones:** Lottie Files
* **Documentación:** Mintlify
